# 📸 FotosessII — Твой личный AI-фотограф!

Добро пожаловать в репозиторий **FotosessII** — передового Telegram-бота, который превращает обычные селфи в профессиональные фотографии с помощью магии нейросетей! 

Вам больше не нужны дорогие студии и фотографы. Загрузите свое фото, опишите желаемый образ (или выберите готовый стиль), и ИИ сгенерирует потрясающий результат за секунды! 🎨✨

## 🌟 Главные фичи

- **🎨 Творческий режим:** Полная свобода действий! Опишите любой образ текстом (киберпанк, фэнтези, строгий костюм) и нейросеть воплотит его в реальность.
- **🚀 Готовые стили:** Лень придумывать? В нашем канале публикуются лучшие работы других пользователей. Понравился стиль? Примените его на себя в один клик!
- **💎 Telegram Stars:** Быстрая, безопасная и нативная оплата через Telegram Stars (XTR) без перехода на сторонние сайты.
- **🛡️ Модерация и Галерея:** Лучшие генерации попадают в наш публичный канал (по согласию пользователя), формируя вдохновляющее комьюнити.
- **💸 Возврат средств:** Если результат не оправдал ожиданий — средства можно вернуть. Мы ценим каждого пользователя!
- **👥 Реферальная система:** Приглашайте друзей по уникальной ссылке и получайте бонусные кредиты за каждую их покупку (1 кредит за каждые 5 купленных другом).

## 🛠️ Технологии

Бот написан с любовью к чистому коду и архитектуре:
- **Aiogram 3.x:** Мощный современный фреймворк для Telegram API.
- **Gemini AI:** Умное описание генераций для постов в канал.
- **Nano Banana 2:** Высококлассная генерация изображений.
- **SQLite / aiosqlite:** Легковесная и быстрая база данных для пользователей и заказов.
- **FSM (Машина состояний):** Надежное ведение пользователя по воронке генерации.

## 🚀 Как запустить

Убедитесь, что у вас установлен Python 3.10+.

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/gdr102/FotosessII.git
   cd FotosessII
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Создайте файл `.env` на основе шаблона и заполните ключи:
   ```env
   TOKEN_TG=ваш_токен_бота
   TOKEN_POLZA=ваш_токен_polza_ai
   ENGINE=midjourney # или другой доступный движок
   ADMIN_REFUND_CHAT_ID=-100... # ID админского чата
   ADMIN_PUBLIC_CHANNEL_ID=-100... # ID канала со стилями
   PAYMENT_MIN_STARS=50
   ```
4. Запустите бота:
   ```bash
   python main.py
   ```
